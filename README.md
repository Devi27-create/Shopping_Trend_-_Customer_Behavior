### A Data Visualization Project Powered by AI Storytelling

## Project Overview
The **Shopping Trends & Customer Behavior Dashboard** is an interactive Power BI report designed to analyze how customers shop â€” what they buy, how often, and under what conditions.  
Using data from 3,900 customers, the dashboard identifies **key behavioral patterns**, **spending habits**, **seasonal trends**, and **marketing opportunities**.

The project demonstrates how **5A's storytelling** can turn raw transactional data into **actionable business intelligence**.

---

## Objectives
- Understand shopping patterns and revenue drivers.
- Identify relationships between **age, gender, season, and product category**.
- Measure the impact of **subscriptions, discounts, and promo codes** on spending.
- Visualize insights that support **marketing, retention, and seasonal strategies**.
- Showcase the use of **5A's in analytics, and data narration**.

---

## Dataset Details
**Source:** Shopping Trends & Customer Behavior Dataset  
**Records:** 3,900  
**Fields Include:**
- Customer demographics (Age, Gender, Location)
- Purchase behavior (Item Purchased, Category, Amount)
- Engagement metrics (Subscription, Promo Code, Discounts)
- Seasonality and Payment Method  
- Review Ratings and Previous Purchases

## Project Framework
This project follows the **5A Model** for data storytelling:

| Stage          | Description                                                              |
|----------------|--------------------------------------------------------------------------|
| **Ask**        | Define analytical questions around shopping trends and customer loyalty. |
| **Analyze**    | Clean and explore data to find meaningful relationships.                 |
| **Articulate** | Transform insights into clear, visual narratives using Power BI.         |
| **Actualize**  | Build an interactive dashboard for real-time exploration.                |
| **Action**     | Translate findings into marketing and product recommendations.           |

## Business Recommendations
- Increase subscription campaigns to grow loyalty (target 40%).  
- Focus marketing budgets on **Fall/Winter** seasons.  
- Promote underperforming categories like **Footwear** and **Outerwear**.  
- Maintain and expand **promo code programs** due to their positive ROI.  
- Improve **customer satisfaction** by addressing low-rated products.
